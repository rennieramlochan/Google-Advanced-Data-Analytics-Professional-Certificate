Google Advanced Data Analytics Professional Certificate
Certificate:

Certificate for Rennie Ramlochan, Google Advanced Data Analytics Professional Certificate, offered through Google.


The focus of this portfolio is to gather, improve, and complete activities related to advanced data analysis, predictive modeling, and machine learning, as presented in Google's Advanced Data Analytics Professional Certificate offered through Coursera. This repository serves as a professional portfolio documenting my completion of the program.

The program consists of seven courses designed to equip learners with essential skills for data science, data analysis, and machine learning. The projects and activities contained herein demonstrate proficiency in Python programming, statistical analysis, regression modeling, machine learning, and data storytelling using the PACE (Plan, Analyze, Construct, Execute) project framework.

Program Overview
Those who earn the Google Advanced Data Analytics Professional Certificate have demonstrated competency in:

Data Science Foundations: Applying the PACE framework to manage end-to-end data projects.

Python Programming: Utilizing data libraries like Pandas, NumPy, and Scikit-learn for advanced manipulation and modeling.

Exploratory Data Analysis (EDA): Identifying patterns, trends, and outliers to drive actionable insights.

Statistical Analysis: Applying probability, sampling, and hypothesis testing to make data-driven decisions.

Regression Analysis: Building and interpreting linear and logistic regression models.

Machine Learning: Implementing and evaluating supervised and unsupervised learning algorithms.

Course Curriculum
Foundations of Data Science
Introduction to data roles, ethics, and the PACE project workflow.
Key Topics: Data project lifecycle, stakeholder communication, and professional data ethics.
Activities: Structuring project workflows and defining business problems.

Get Started with Python
Building core programming skills for data analysis.
Key Topics: Python syntax, data structures, and automation scripting.
Activities: Cleaning datasets and performing initial data wrangling.

Translate Data into Insights
Turning raw data into meaningful visualizations.
Key Topics: Exploratory Data Analysis (EDA) and data storytelling.
Activities: Creating visualizations in Tableau and conducting deep-dive EDA.

The Power of Statistics
Statistical methods for data-informed decision-making.
Key Topics: Probability distributions, confidence intervals, and hypothesis testing.
Activities: Conducting A/B tests and statistical inference.

Regression Analysis
Predictive modeling using regression techniques.
Key Topics: Simple and multiple linear regression, logistic regression.
Activities: Building models to predict business outcomes.

The Nuts and Bolts of Machine Learning
Implementation of predictive algorithms.
Key Topics: Random Forest, XGBoost, and model evaluation metrics.
Activities: Training, tuning, and validating machine learning models.

Google Advanced Data Analytics Capstone
Integrated application of course skills to solve a complex real-world problem.
Key Topics: Full-cycle project deployment and technical communication.
Activities: Building an end-to-end analytics pipeline from problem identification to final stakeholder presentation.

Repository Structure
